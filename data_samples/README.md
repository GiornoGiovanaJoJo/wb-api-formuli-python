# Примеры данных

В этой папке находятся примеры CSV файлов с данными из Wildberries.

## Файлы:

### 1. `43-nedelia-2-List1.csv`
Пример отчёта WB за неделю 43.

**Основные поля:**
- Артикул ВБ (nm_id)
- Доставки, Отказы, Продажи
- Возвраты, Самовыкупы, Раздачи
- Продажи до СПП и после СПП
- Логистика, Хранение, Комиссия
- Штрафы, Доплаты
- Реклама (ДРР)
- Себестоимость
- Операционная прибыль

### 2. `parsed_wb_report.csv`
Обработанный отчёт WB в удобном формате.

## Как использовать:

```python
from data.loader import DataLoader

# Загрузить CSV
data = DataLoader.load_from_csv('data_samples/43-nedelia-2-List1.csv')

# Парсинг в модели
from models.product import ManualInputData

manual_data = ManualInputData(
    cost_per_unit=2032,
    self_purchase_count=1,
    giveaway_count=0
)

product = DataLoader.parse_wb_csv_to_product(data[0], manual_data)
```

## Структура CSV:

### Движение товара (шт):
- Доставки
- Отказы
- Продажи
- Самовыкупы
- Раздачи
- Возвраты

### Финансы (руб):
- Продажи до СПП
- Продажи после СПП
- Сумма СПП
- К перечислению за товар

### Расходы WB (руб):
- Фактическая комиссия
- Логистика
- Штрафы
- Доплаты
- Хранение
- Платная приёмка
- Реклама (баланс + счет)
- Прочие удержания

### Итоги:
- Итого к оплате
- Налог (НДС)
- Себестоимость
- Операционная прибыль
- ROI
